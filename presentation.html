Vision

One app. Three AI providers. The cleanest mobile chat experience on the market.

🎯 What it is

A polished mobile AI chat app that puts Claude, GPT, and Gemini in a single inbox — no tab switching, no three subscriptions.

👤 Who it's for

General consumers who already use AI daily and want a faster, cleaner, mobile-first experience.

🏆 Why it wins

Executes the established pattern better: faster streaming, multi-provider access, and a trustworthy privacy model.

Target User — Jobs To Be Done

  • Get a fast, capable answer without switching between three apps
  • Continue a conversation later from any device
  • Ask about an image or document by attaching it directly
  • Dictate a question hands-free while commuting
  • Control which model answers when one works better for the task
  • Trust that conversations are private and deletable
  • Stay within budget with a clear usage meter

Non-Users (v1)

No enterprise / SSO No team workspaces No web search grounding No code execution No API access

Non-Functional Requirements

⚡ Performance

  • First token P95 < 3s
  • Cold start < 3s
  • Conversation list < 1s (500 convos)
  • Offline: read cached convos

🔒 Security

  • API keys backend-only
  • TLS 1.3 minimum
  • Argon2id password hashing
  • Rate limiting on all endpoints
  • JWT 15-min expiry + rotation

📈 Scalability

  • Stateless, horizontally scalable
  • 99.5% monthly uptime SLA
  • Provider error → clear fallback

♿ Accessibility

  • WCAG 2.1 AA contrast
  • VoiceOver + TalkBack labels
  • 44×44pt min touch targets
  • Voice features are additive
GDPR Compliant CCPA Compliant App Store Guidelines Play Store Policy Age Rating: 12+

Open Questions & Assumptions

Open Questions

  • App name — "claude-codex-chat" is a working title
  • Annual pricing — $89.99/yr needs pricing strategy confirm
  • Push notifications — iOS BGAppRefreshTask constraints
  • File extraction library — DOCX / PDF server-side tool TBD
  • Malware scanning — ClamAV latency impact TBD
  • Data export delivery — async email vs in-app download
  • 30-day free history limit — right balance to drive upgrades?
  • Provider failover — manual-only in v1 acceptable?

Key Assumptions

  • Provider latency stays within 3-second P95 budget
  • 5-image per message limit consistent with provider APIs
  • 7-day soft-delete with no undo UI acceptable
  • 30-day GDPR erasure window OK given backup retention
  • Client-side syntax highlighting adds <200 KB
  • Free-user blended AI cost manageable at launch scale
  • Training data opt-out default; opt-in available later
  • v1 English only; localization deferred to v2
